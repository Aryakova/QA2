Тест-кейс 1: Позитивная проверка авторизации (валидный логин и пароль)

Предусловие:

    Открыть сайт https://www.saucedemo.com/

    Поля Username и Password пустые

Шаги:

    Ввести валидный логин (например, standard_user) в поле Username.

    Ввести валидный пароль (например, secret_sauce) в поле Password.

    Нажать кнопку Login.

Ожидаемый результат:

    Происходит успешная авторизация.

    Открывается главная страница с товарами (Products).

    В шапке страницы отображается заголовок Products.
    
 Тест-кейс 2: Негативная проверка авторизации (неверный пароль)

Предусловие:

    Открыть сайт https://www.saucedemo.com/

    Поля Username и Password пустые

Шаги:

    Ввести валидный логин (например, standard_user) в поле Username.

    Ввести неверный пароль (например, wrong_password) в поле Password.

    Нажать кнопку Login.

Ожидаемый результат:

    Появляется сообщение об ошибке: Epic sadface: Username and password do not match any user in this service.

    Авторизация не происходит, пользователь остается на странице входа.

Тест-кейс 3: Негативная проверка (попытка входа с пустыми полями)

Предусловие:

    Открыть сайт https://www.saucedemo.com/

    Поля Username и Password пустые

Шаги:

    Оставить поле Username пустым.

    Оставить поле Password пустым.

    Нажать кнопку Login.

Ожидаемый результат:

    Появляется сообщение об ошибке: Epic sadface: Username is required.

    Авторизация не происходит.
